题目1：基本输出 [简单]

编写一个程序，输出以下信息：

姓名：你的名字
年龄：你的年龄
专业：二进制工程

要求：使用printf函数，格式美观。

题目2：数据类型和大小 [简单]

编写程序，打印以下数据类型的大小（字节数）：

  • char
  • short
  • int
  • long
  • float
  • double

要求：使用sizeof运算符。

题目3：变量计算 [简单]

编写程序：

  1. 声明三个整数变量：a = 15, b = 7, c = 3
  2. 计算并输出：
    • a + b - c
    • a * b / c
    • a % b
    • (a + b) * c

题目4：位运算练习 [中等]

编写程序，完成以下位运算：

  1. 定义两个无符号整数：a = 0b11001100, b = 0b11110000
  2. 计算并输出（十六进制格式）：
    • a & b（按位与）
    • a | b（按位或）
    • a ^ b（按位异或）
    • ~a（按位取反）
    • a << 2（左移2位）
    • a >> 2（右移2位）

提示：使用 printf("0x%x\n", value) 输出十六进制。

题目5：输入输出 [简单]

编写一个程序：

  1. 提示用户输入两个整数
  2. 读取这两个整数
  3. 计算它们的和、差、积、商（整数除法）
  4. 格式化输出结果

示例输出：

请输入第一个整数: 20
请输入第二个整数: 5
20 + 5 = 25
20 - 5 = 15
20 * 5 = 100
20 / 5 = 4

题目9：位标志操作 [困难]

实现一个简单的标志系统：

  1. 定义三个标志位：
    • FLAG_READ = 0x01 (二进制: 00000001)
    • FLAG_WRITE = 0x02 (二进制: 00000010)
    • FLAG_EXECUTE = 0x04 (二进制: 00000100)
  2. 创建一个变量flags，初始为0
  3. 实现以下操作：
    • 设置READ标志
    • 设置WRITE标志
    • 检查是否有READ权限
    • 清除WRITE标志
    • 切换EXECUTE标志
  4. 每次操作后打印flags的值（二进制和十六进制）

提示：使用位运算符 |, &, ^, ~

题目10：综合练习 [困难]

编写一个“学生信息管理系统“的简单版本：

    • 年龄（整数）
    • 成绩（浮点数）
  3. 格式化输出学生信息，包括：
    • 姓名（左对齐，20字符宽度）
    • 年龄（右对齐，5字符宽度）
    • 成绩（保留2位小数）
  4. 计算并输出：
    • 成绩的十六进制表示（将浮点数转换为整数后）
    • 年龄的二进制表示（自己实现转换）

题目9参考答案

#include <stdio.h>

#define FLAG_READ    0x01  // 00000001
#define FLAG_WRITE   0x02  // 00000010
#define FLAG_EXECUTE 0x04  // 00000100

void print_binary(unsigned int n) {
    for (int i = 7; i >= 0; i--) {
        printf("%d", (n >> i) & 1);
    }
}

int main() {
    unsigned int flags = 0;
    
    printf("初始flags: 0x%02x (二进制: ", flags);
    print_binary(flags);
    printf(")\n");
    
    // 设置READ标志
    flags |= FLAG_READ;
    printf("设置READ后: 0x%02x (二进制: ", flags);
    print_binary(flags);
    printf(")\n");
    
    // 设置WRITE标志
    flags |= FLAG_WRITE;
    printf("设置WRITE后: 0x%02x (二进制: ", flags);
    print_binary(flags);
    printf(")\n");
    
    // 检查READ权限
    if (flags & FLAG_READ) {
        printf("有READ权限\n");
    }
    
    // 清除WRITE标志
    flags &= ~FLAG_WRITE;
    printf("清除WRITE后: 0x%02x (二进制: ", flags);
    print_binary(flags);
    printf(")\n");
    
    // 切换EXECUTE标志
    flags ^= FLAG_EXECUTE;
    printf("切换EXECUTE后: 0x%02x (二进制: ", flags);
    print_binary(flags);
    printf(")\n");
    
    return 0;
}

题目10参考答案

#include <stdio.h>

#define MAX_NAME_LEN 50

void print_binary(int n) {
    if (n == 0) {
        printf("0");
        return;
    }
    int bits[32];
    int count = 0;
    while (n > 0) {
        bits[count++] = n % 2;
        n /= 2;
    }
    for (int i = count - 1; i >= 0; i--) {
        printf("%d", bits[i]);
    }
}

int main() {
    char name[MAX_NAME_LEN];
    int age;
    float score;
    
    printf("请输入学生姓名: ");
    scanf("%s", name);
    
    printf("请输入年龄: ");
    scanf("%d", &age);
    
    printf("请输入成绩: ");
    scanf("%f", &score);
    
    printf("\n=== 学生信息 ===\n");
    printf("姓名: %-20s\n", name);
    printf("年龄: %5d\n", age);
    printf("成绩: %.2f\n", score);
    
    // 成绩的十六进制（转换为整数）
    int score_int = (int)score;
    printf("成绩(十六进制): 0x%x\n", score_int);
    
    // 年龄的二进制
    printf("年龄(二进制): ");
    print_binary(age);
    printf("\n");
    
    return 0;
}
