# Binary Engineer Learning

> AI驱动的二进制工程师和系统编程完整学习路径

欢迎！这是一个交互式学习平台，你可以**边看边练习**，直接在浏览器中运行代码。

## 特性

- **交互式学习**：每个章节都有可运行的代码示例
- **在线练习**：直接在浏览器中编辑和运行代码
- **即时反馈**：运行代码立即看到结果
- **循序渐进**：从基础到进阶，系统学习
- **AI教学**：完全由AI生成和维护的学习材料

## 学习路径

1. **C语言基础** - 掌握底层编程基础
2. **Python基础** - 快速开发和脚本编写
3. **数据结构与算法** - 建立算法思维
4. **C++进阶** - 面向对象和系统编程
5. **汇编语言** - 理解底层执行机制
6. **逆向工程** - 二进制分析和安全技术

## 使用说明

- 每个章节包含**理论讲解**和**练习题**
- 代码块可以直接编辑和运行
